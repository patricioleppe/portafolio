# Patricio Leppe - Portfolio

Portfolio personal desarrollado con **Django** y **PostgreSQL**, donde presento mi experiencia profesional, proyectos, tecnologías y trayectoria como desarrollador Backend especializado en Python.

## Sobre el Proyecto

Este sitio fue construido como una aplicación web autoadministrable utilizando Django y está desplegado en Fly.io.

Permite mostrar:

* Información profesional y experiencia laboral.
* Proyectos destacados.
* Tecnologías y herramientas utilizadas.
* Formulario de contacto.
* Descarga de CV.
* Gestión dinámica de contenido mediante Django.

## Ejecución Local

```bash
git clone https://github.com/patricioleppe/portafolio.git

cd portafolio

python -m venv .venv

source .venv/bin/activate
# Windows
# .venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
